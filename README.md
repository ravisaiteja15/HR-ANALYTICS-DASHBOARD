# HR ANALYTICS DASHBOARD

### 📌Project Overview
This project analyzes employee data to identify factors affecting employee performance, satisfaction, and attrition. The goal is to provide data-driven insights that help organizations improve employee retention and workforce planning.

---

### Objectives
- Analyze employee demographics
- Study employee attrition patterns
- Identify key factors influencing employee turnover
- Visualize HR metrics using charts and dashboards
- Build predictive models for employee attrition

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

### Dataset
The dataset contains employee information such as:
- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Job Satisfaction
- Performance Rating
- Attrition
