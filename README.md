# HR ANALYTICS DASHBOARD

### 📌Project Overview
Developed an HR Attrition Analytics Dashboard in Power BI to analyze employee turnover patterns and identify factors influencing attrition. Created interactive visualizations and KPIs to analyze attrition trends based on department, job role, salary, age, gender, and experience. Applied data cleaning, data modeling, and DAX calculations to generate actionable workforce insights.

---

### 🚀Objectives
- Analyze employee demographics
- Study employee attrition patterns
- Identify key factors influencing employee turnover
- Visualize HR metrics using charts and dashboards
- Build predictive models for employee attrition

---

### 🛠️ Tools & Technologies Used
- 📊 Power BI
- 🧹 Power Query (Data Cleaning & Transformation)
- 📐 DAX (Data Analysis Expressions)
- 📁 Excel / CSV files
- 📈 Data Visualization

---

### 📂Dataset Information
- **Source :** Kaggle HR Analytics Dataset
- Contains employee-related information covering demographics, job roles, salary details, work experience, performance, satisfaction levels, and attrition status.
 - Provides data points to identify factors influencing employee turnover and workforce trends.
